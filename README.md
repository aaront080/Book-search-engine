## Description

This application is a Book Search Engine using the google books API and GraphAPI with Apollo Server. This application was created using a MERN stack. Users can sign up and log back into an account and use the application to search for books and add books to a wish list. 


## Installation

- To use this app npm init has to be called to create your package.json file.
- Run an NPM install to install the necessary dependencies.
- Run "npm start" for the port to start listening with mongoose.
